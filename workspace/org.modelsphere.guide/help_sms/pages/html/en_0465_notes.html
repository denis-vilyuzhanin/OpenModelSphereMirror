Using graphical notes


A graphical note is an annotation to add a comment in a graphical representation. These notes can be graphically linked to any model element which has a graphical representation.

In oppose to comments / descriptions within the properties of a model element these annotations can be used for temporary remarks which do not have any relevance anymore once the model / diagram has been finished.

To add a graphical note to a diagram:

1. Select the notice tool from the Create Tools toolbar .
