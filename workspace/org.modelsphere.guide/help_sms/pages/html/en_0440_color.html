Color and line styles


Using colors and line styles to enhance models

Open ModelSphere offers a choice of graphical effects for powerful model presentation. One of the graphical detail that you can use to enhance model aesthetics and impart additional information is color. Colors for font, border and text can be modified. Another graphical details that you can use is line style. Line style can be modified from plain into a dash line or highlight.

To modify the object format:
  1. Right-click the selected object and look for the object Format command.
  2. Under the Color tab, click inside the squares to change colors for background, border or text.
  3. Under the Line Style tab, check the boxes for dash line and highlight, if wanted.


These features are also available in the Format toolbar or under Format in the main menu.




Transparency

Open ModelSphere supports transparency in the diagrams. You can set this parameter in the color selector.
