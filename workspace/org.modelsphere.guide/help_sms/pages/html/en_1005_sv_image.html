Generating a diagram's image file


A diagram’s image can be used for multiple purposes (Web publishing, including diagrams in a word processor document, etc.). Therefore, Open ModelSphere supports to save the diagram images.

To save a diagram in picture format:
  1. Select the diagram (highlight it)
  2. From the main menu, choose File > Save Diagram as Image.
  3. Choose your options and settings.
  4. Click Save.
The file (picture of the diagram) will be saved in a .jpg format.

You can also select the diagram in the explorer, right-click it and from the pop-up menu choose Save Diagram as Image.




Options and settings available

Before generating the JPEG version of the diagram, you need to choose between default settings or customized ones. Here are the possibilities:
  • One File Only: Creates one image file (.jpeg) containing all diagram pages. The name of the file will be formatted as: [diagram name] + ".jpg".

  • Multiple Files: Creates one image file (.jpeg) per page. It is possible to select the range of pages to save as image. The name of the file will be formatted as: [diagram name]+ "_" + [page number] + ".jpg".

  • Image scale: Sets the scaling factor for the generated image. Default image scale is 75%.

  • JPEG Quality: Sets the quality of the JPEG image file. Default quality is 60%.

  • File: Sets the directory where the file(s) will be saved. Use Browse to navigate to the folder you want.
