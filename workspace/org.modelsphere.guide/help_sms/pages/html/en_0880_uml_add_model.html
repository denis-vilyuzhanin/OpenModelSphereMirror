Building a UML Model


It is possible to create a UML model from the main menu, from the explorer window and from the Edit toolbar.
  • From the main menu: choose Edit > Add > UML Model.

  • From the explorer window: right-click project. Then, from the pop-up menu, choose Add > UML Model and click on one of the UML models.

  • From the Edit toolbar: click the Add tool and choose UML Model.

At the model level there is no diagram yet. Since process models can contain more than one systems, diagrams are added at the system level.
