The status bar


The status bar is located at the bottom of your screen. It gives useful information regarding:
  • message: displays relevant application information, such as opening or saving operations and tooltips.
  • project: displays the active project’s name and location.
  • database: displays RAM, repository information and an enabled or disabled database icon.
It is possible to display or hide the status bar.
For this option, choose in the main menu : Display / Hide/Show status bar.
