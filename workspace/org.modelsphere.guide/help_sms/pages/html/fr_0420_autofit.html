Ajuster automatiquement


Par défaut, open ModelSphere ajustera toujours les objets graphiques (sauf pour les objets du modèle de processus), maximisant ainsi l’affichage et l’espace utilisé. Si vous désirez enlever cette option, vous devez cliquer avec le bouton droit sur l'objet graphique en question et décocher l'item Ajustement automatique.
