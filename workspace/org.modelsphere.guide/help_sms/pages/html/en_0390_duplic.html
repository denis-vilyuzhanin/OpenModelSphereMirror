In Open ModelSphere, you have the option to create a graphic object from the pop-up menu of:
  • a semantic object (i.e. explorer selection)
  • a graphic object (i.e. diagram selection)
The target is always a diagram and it must be opened as the new graphic objects will be created in the center of the window.


Moreover, the following limitations apply to the list of available target diagrams:

Class diagrams
  • Any Java class diagram of the project
  • Any Java package diagram of the project
Other diagrams
  • Any diagram of the same model is accepted
