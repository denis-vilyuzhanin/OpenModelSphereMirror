Modifier la multiplicité


La multiplicité est une indication sur la gamme de cardinalités qu'un ensemble peut assumer. Des indications de multiplicité peuvent être définies pour des rôles dans les associations, des parties avec des composites, des répétitions ou autres. Essentiellement, une multiplicité est un sous-ensemble (possiblement infini) de nombres entiers non négatifs.

Lorsqu’une association est créée, les valeurs par défaut 0,N et 1,1 sont attribuées aux entitées sources et cibles respectivement. Toutefois, il est possible de changer le facteur de multiplicité dans l'association.
  1. Placez le pointeur sur le label de description du rôle (ou association) et cliquez avec le bouton droit.
  2. Dans le menu contextuel, choisir Multiplicité, puis l'option désirée.


Vous pouvez également changer la multiplicité de l’association à partir du panneau de conception. Cliquez à gauche sur le label de description du rôle (ou association) et éditez la ligne Multiplicité.
