Grouper les diagrammes


Il est possible de grouper deux diagrammes ou plus dans le but de former un seul diagramme. Le format de chaque diagramme sera gardé et la notation peut être choisie dans l'éventualité où les diagrammes n'ont pas la même notation.

Pour grouper les diagrammes :
  1. Sélectionnez les diagrammes à être consolidés et appuyez sur la touche Ctrl.
  2. Cliquez avec le bouton droit sur les diagrammes.
  3. Sélectionnez Grouper les Diagrammes.

Or, you can do the following:
  1. Sélectionnez les diagrammes à être consolidés et appuyez sur la touche Ctrl.
  2. Outils > Grouper les Diagrammes..
NOTE:  Il est aussi possible d'invoquer l'option Grouper les Diagrammes en sélectionnant un modèle. Ceci aura pour effet de grouper tous les diagrammes se trouvant dans le modèle sélectionné.
