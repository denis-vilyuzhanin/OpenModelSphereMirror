Convertir un modèle relationnel de données en modèle conceptuel de données

Un modèle relationnel peut être converti en modèle conceptuel. Les tables sont transformées en entités, les connecteurs en associations etc..

Il est possible de convertir un modèle relationnel de données à partir du menu principal, de la fenêtre Explorateur et du diagramme.

  • Du menu principal, choisir Outils > Convertir en Mode Conceptuel....
  • De la fenêtre Explorateur, cliquer à droite sur un modèle (ou le diagramme du modèle). Un menu contextuel aparaîtra. De ce dernier, choisir Convertir en Mode Conceptuel....
  • Du diagramme, cliquer à droite,choisir Convertir en Mode Conceptuel....



Le processus de conversion crée un nouveau modèle. Vous pouvez choisir

  • De seulement convertir un modèle dans le diagramme actif ou inclure tous les sous-modèles
  • La notation pour le modèle du cible
  • Absorber les associations optionnelles

Veuillez noter qu’il faut

  • Épurer le modèle
  • Vérifier l’intégrité
avant la conversion.

Un modèle relationnel et ses modèles convertis sont indépendants les uns des autres, ce qui signifie que les modifications faites après la conversion ne se réflèteront pas dans le modèle converti.
