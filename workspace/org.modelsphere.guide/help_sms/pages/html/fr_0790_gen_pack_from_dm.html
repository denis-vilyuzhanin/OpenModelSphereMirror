Génération de paquetage à partir d'un modèle de données


Afin d'accomplir cette opération, vous devez sélectionner un modèle de données. Si vous souhaitez créer des liens, vous devrez également avoir un modèle de liens.
  1. Ajoutez un modèle de classes ou un paquetage (si vous n’en avez pas déjà).

  2. Avec le bouton droit, choisissez le modèle de données > Générer un paquetage. La fenêtre Génération d’un modèle de classe apparaît.



  3. Maintenant vous devez choisir l'emplacement du paquetage Java: modèle de classes ou paquetage. En cliquant la boîte, un choix apparaîtra.

  4. Vous pouvez créer des liens. Pour ce faire, cochez la case Créer des liens et choisissez un modèle de liens.
    Rappelez-vous, un modèle de liens est requis si l’option est choisie.
    • Table (source) à la classe (cible).
    • Colonne (source) au champ (cible).


  5. Déterminez la visibilité des champs de classes générés.

  6. Cliquez sur le bouton Générer.



Les objets générés

Paquetage du modèle de données
Le paquetage contenant les classes, les champs et les associations générés.

Paquetage des types de données
Le paquetage contenant les classes générées qui correspondent au type de la colonne.

Objets liés
Si l'option de relation est sélectionnée, les objets liés sont générés dans le modèle de liens choisi.
