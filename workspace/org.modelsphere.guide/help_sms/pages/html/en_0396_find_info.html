Finding information


Finding semantic information

It is possible to find semantic information regarding the graphic objects in your diagram window. Right-click the object and select the Find it in explorer from the popup menu. The selected item will be highlighted in the explorer window.



Finding graphic information

It is also possible to find graphic information regarding the semantic elements in the explorer window. Right-click the word in the explorer window and select Find it in diagram from the pop-up menu. The selected item will be highlighted in the diagram window.
