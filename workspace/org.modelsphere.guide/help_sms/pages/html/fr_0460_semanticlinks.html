Créer des liens sémantiques et graphiques

Un lien sémantique est un concept puissant qui permet de lier n'importe quel objet sémantique avec n'importe quel autre objet sémantique; par exemple on peut lier un dépôt dans un modèle de processus à une table d'un modèle de données ou à une classe d'un modèle de classes UML. Un lien peut aussi avoir plusieurs objets source et plusieurs objets destination.

Les liens sont particulièrement utiles pour relier des éléments du modèle de processus, de données et de classes entre eux. Dans le cas où les éléments liés ont des représentations graphiques dans le même diagramme, on peut afficher graphiquement le lien sémantique.
