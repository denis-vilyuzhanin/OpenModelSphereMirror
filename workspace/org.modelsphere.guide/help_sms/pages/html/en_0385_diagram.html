The Diagrams


Depending on the kind of model that you plan to use in ModelSphere, you can add one or many corresponding diagrams that let you graphically create and organize your model.

Data Modeling Diagram
Process Diagram
Class Diagram
Domain Diagram
Common Item Diagram


To add a diagram to a model:
  1. Select the model concept in the explorer.

  2. Right-click on it and choose Add > Diagram. Or click on Add in the toolbar and choose Diagram.



Note:
Process diagrams are only available under the context and the process concept (not under the model).
The class diagrams are also available under the package concept.
It's possible to set automatic creation of the diagrams when a model is created. See Modifying the options.



Empty diagram...

If you add a diagram, it will be empty even if objects already exist in the model. To display them, choose Add missing Diagram Objects in the pop-up menu of the diagram.
