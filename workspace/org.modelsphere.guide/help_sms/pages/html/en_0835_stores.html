Adding stores


The store’s graphic representation displays the name, the identifier, and the alias of each of the model stores. You can also switch from a standard store to a control store (displayed with a dotted line on the diagram window) by checking the Control box in the Properties window...

To add a store:
  1. Select the store creation tool located in the Process modeling toolbar.

  2. With the store creation tool selected, click anywhere inside the diagram window.

The Properties window of the store contains a menu that gives information regarding the store and linked objects.



Properties

The Design tab of the Properties window provides the following information:
  • the store name

  • its numerical identifier: which is automatically given when you create a new store but can be modified directly in this window or by renumbering

  • its alias: that you can use to display another name

  • its physical name

  • its volume

  • a description

  • and if it is a control store.
